- The module is organized into two logical layers: a `lib/` layer containing pure utility modules (`api.js` for HTTP/WebSocket connectivity with token management, `tauri.js` for safe Tauri IPC invocation and desktop notifications) and a `components/` layer providing React UI primitives (`NotificationSystem.jsx` with context-based toast/inbox management, `DownloadProgress.jsx` listening to Tauri events).
- `catalog.js` acts as a static data module defining item rarity tiers and the in-app library catalog, consumed by pages rather than providing runtime logic.
- Dependency direction flows from UI components toward lib utilities (e.g., `NotificationSystem` imports `notifyDesktop` from `tauri.js`; `DownloadProgress` imports `listen` from `tauri.js`), keeping infrastructure concerns decoupled from page-level routing.
- `index.css` establishes global Tailwind-based styling conventions (custom scrollbar, hidden cursor, base animations) that apply across the entire application scope.
