- Centralized State & Routing: `App.jsx` manages global authentication state via `localStorage` and conditionally renders the `LoginPage` or `MainLayout`, acting as the primary entry point for all child modules.
- Shared Infrastructure Wiring: Child modules consume shared utilities from `src/lib/` (`api.js` for authenticated HTTP requests, `tauri.js` for native IPC invocations, and `window.js` for window controls), ensuring consistent interaction with the backend and OS.
- Real-time Event Hub: `MainLayout.jsx` establishes a persistent WebSocket connection that dispatches real-time updates (balance changes, friend requests, support tickets) to both the in-app `NotificationSystem` and native Tauri desktop notifications, bridging social, commerce, and support domains.
- Native Integration Bridge: The layout exposes global window methods (`__navigateTo`, `__launchGame`) that allow the Rust backend (tray/menu) to command the frontend, while simultaneously pushing UI state (notifications, user balance) back to the tray via `invoke`.
