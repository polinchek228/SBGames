# Редизайн билдера сборок — карточки, статистика, обратная связь

## Файл
`src/pages/PlayPage.jsx` — единственное изменяемое место.

## Что меняем

### 1. Карточный лейаут (5 карточек)
Вместо одного длинного списка — визуально разделённые секции:

| Карточка | Содержимое | Заголовок |
|----------|-----------|-----------|
| **A — Шапка + статистика** | Заголовок, кнопка «Новая», ряд статистики (моды/паки/шейдеры) | — |
| **B — Настройки** | Профили, название, версия + загрузчик | «Настройки» |
| **C — Контент** | Установленные моды/паки/шейдеры со списком + история запусков | «Контент» |
| **D — Импорт** | Архив (.zip/.jar) + импорт по ссылке | «Импорт» |
| **E — Действия** | Кнопки действий (обновить, бэкап, экспортировать, сохранить профиль) + инфо-заметка | «Действия» |

Каждая карточка:
```
rounded-xl p-3.5
background: rgba(255,255,255,0.03)
border: 1px solid rgba(255,255,255,0.06)
```
Заголовок карточки:
```
text-[9px] uppercase tracking-widest font-semibold
color: rgba(255,255,255,0.3), mb-2
```

### 2. Статистика (Карточка A)
Под заголовком — ряд из 3 мини-ячеек:

| Ячейка | Иконка | Цвет | Источник |
|--------|--------|------|----------|
| Модов | Package | #60a5fa | `draft.mods.filter(m => !m.disabled).length` |
| Паков | FileImage | #a78bfa | `draft.resourcePacks.filter(m => !m.disabled).length` |
| Шейдеров | Sparkles | #f472b6 | `draft.shaders.filter(m => !m.disabled).length` |

Каждая ячейка: `rounded-lg px-2 py-2 text-center`, bg `${color}0a`, border `${color}18`. Число в акцентном цвете `text-[14px] font-bold`, подпись `text-[8px] uppercase`.

### 3. Кнопки действий (Карточка E)
**Основные** (полная ширина, градиент):
- «Обновить всё» — синий градиент + glow, `h-9 rounded-xl text-[11px] font-bold`
- «Сохранить профиль» — аналогично

**Вторичные** (в ряд, приглушённые):
- «Бэкап» + «Экспорт» — `flex-1 h-8 rounded-lg`, bg `rgba(255,255,255,0.04)`

Инфо-заметка внизу карточки через `border-top`.

### 4. Обратная связь
- **saveFlash** — после сохранения профиля кнопка мигает зелёным 800мс
- **importFlash** — после импорта карточка D подсвечивается зелёной рамкой 1200мс
- **Карточки** — `motion.div` с `initial={{ opacity: 0, y: 6 }}`, `animate={{ opacity: 1, y: 0 }}`, каскадная задержка `index * 0.05`

### 5. Два новых стейта
```jsx
const [saveFlash, setSaveFlash] = useState(false);
const [importFlash, setImportFlash] = useState(false);
```

## Шаги реализации

1. **Добавить стейты** `saveFlash`, `importFlash` (~строка 124)
2. **Добавить `setSaveFlash(true)` + setTimeout** в `saveProfile` и `saveModpack`
3. **Добавить `setImportFlash(true)` + setTimeout** в обработчик архивного импорта и `importFromUrl`
4. **Заменить контейнер** (строки 1019–1249) на карточную структуру:
   - Внешний wrapper: `p-4 gap-3 flex-1 overflow-y-auto`
   - Card A: заголовок + статистика
   - Card B: настройки (профили + название + версия/загрузчик)
   - Card C: контент (установленные моды + история)
   - Card D: импорт (архив + ссылка) с `importFlash`
   - Card E: действия (primary/secondary кнопки + инфо) с `saveFlash`
5. **Обернуть каждую карточку** в `motion.div` для анимации появления
6. **Проверить**, что правая панель (строки 1261+) не затронута

## Верификация
- `npm run dev` → открыть билдер → проверить визуально
- Создать сборку, добавить моды из Modrinth → статистика обновляется
- Импортировать .jar → зелёная подсветка карточки
- Сохранить профиль → зелёная вспышка кнопки
- Обновить моды → спиннер работает
- Бэкап / Экспорт → функции работают
- Правая панель Modrinth — не изменилась
